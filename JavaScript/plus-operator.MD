# The `+` Operator
Binary operator တစ်ခုဖြစ်တဲ့ + operator ဟာ
- numeric operand တွေကို add လုပ်ပေးသလို
- string operand တွေကို concatenate လုပ်ပေးပါတယ်။

`1+2` // => 3 [^1]

`"1" + "2"` // => '12'

`"hello" + " " + "Mg Htoo"` // => "hello Mg Htoo"
